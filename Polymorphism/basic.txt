Polymorphism means:

One name having many forms (or behaviors).

In C++ this can be achieved using:

1. Function Overloading
2. Operator Overloading
3. Virtual Functions (Run-time polymorphism)
1️⃣ Compile-Time Polymorphism

Supported by:

Function Overloading
Operator Overloading

Example:

int add(int a,int b);
float add(float a,float b);

Same function name:

add()

Different forms → polymorphism.

2️⃣ Operator Overloading Example
Complex c1, c2;
c1 + c2;

Here:

+ operator

works differently for objects, which is also polymorphism.

3️⃣ Run-Time Polymorphism

Supported by:

Virtual Functions

Example:

Base pointer → Derived object

Function decision happens during execution.
